# Ocass
OCaSS = Open Source Compliance and Security Scanning

🛠️ Install Syft
In Windows System CLI :

iwr -useb https://raw.githubusercontent.com/anchore/syft/main/install.ps1 | iex

🛠️ Install Syft
In github codespace

curl -sSfL https://raw.githubusercontent.com/anchore/syft/main/install.sh | sh

Afte installation:

Check the version:

syft version
